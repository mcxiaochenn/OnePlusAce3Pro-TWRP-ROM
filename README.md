### 使用教程

1. 前往 Release 并找到你需要的系统版本
2. 下载除 Source Code(xxx) 外的所有文件（应为数个 rom.z0x 和 Merge.sh）
3. 重启手机进入TWRP卡刷    请使用酷安@桜酱没有未来（[GitHub](https://github.com/SakuraKyuo)）制作的TWRP
4. 重启即可
